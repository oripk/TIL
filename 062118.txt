외부에서 작성된 프로시져나 프라퍼티를 사용하려면

import(키워드) + 외부 파일이나 모듈 이름 
위 문장을 작성하여 컴파일러에게 이를 사용한다고 알리고

실제 사용하기 위해서는 이름.원하는 기능 
이렇게 작성하여 기능을 이용한다.
ex) import math
math.pi = 3.141592~

dir 키워드 : 해당 모듈에 존재하는 프로시져나 프라퍼티를 보여준다.

__name__ : 실제 해당 모듈의 이름을 반환하는 프로시져

as : 이전 이름을 이후 이름으로 사용
ex) import my_code as import_module


####### 코딩 스타일 #########

docstring

ex)
"""

"""

모듈을 설명하는 것은 파일의 맨 꼭대기에 위치해야함 
클래스 정의 아래에는 클래스의 목적이 적혀있어야함
함수 밑에도 함수의 목적이 적혀있어야 함

-> help() : 프로시져는 인자와 관련된, 위에서 적은 설명들을 알려줌

comment : 
ex) # 으로 시작

전역상수(모든 글자가 대문자인 이름)는 가능하나, 전역 변수는 절대 사용해서는 안된다.

이름
	변수이름은 소문자로 시작
	프로시져나 메소드 이름은 항상 소문자로 시작(대문자 대신 언더바 사용)
	클래스 이름은 항상 대문자 사용 그리고 언더바 사용 금지 ex) SomeClassName

클래스나 인스턴스의 항목을 사용할때는 getter 함수를 사용하자 혹은 self 키워드 노테이션
